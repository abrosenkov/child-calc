# Child vaccination date calculator

Small static web app for calculating a child's vaccination schedule from their birth date. The user can enter the date manually or choose it from the calendar picker, and the app shows the expected vaccination dates, current status, and the nearest upcoming vaccination.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python -m http.server 4173
```

Then open `http://localhost:4173`.
